1.1.4 First Example

Statements & Expressions. Python code consists of expressions and statements. Broadly, computer programs consist of instructions to either

  1. Compute some value
  2. Carry out some action

Statements typically describe actions. When the Python interpreter executes a statement, it carries out the corresponding action. On the other hand, expressions typically describe computations. When Python evaluates an expression, it computes the value of that expression. This chapter introduces several types of statements and expressions.

This is an article about how to build a personal website with Github and Hexo framwork

There are some folders you may need to know:
scaffolds:
when you new a file, for example, hexo new filename, the file will have elements you already set such as title date tags, etc.

souce:
this could be the folder you may get in touch most, you write sth in it.

theme:
make you page look nice

butterfly
Matery
flex-block

node_modules:
the hexo runs on the node.js enviornment, don’t touch it anyway

public:
this is code you pull to github repo

some basic command you may want to know:
hexo g
hexo s
hexo d
hexo new
hexo publish nameOfTheDraft
hexo new page c - create a normal page c
